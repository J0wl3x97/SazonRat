# Como instalar

Instale o python 3.8, quando for instalar não esqueça de selecionar "ADD PATH PYTHON 3x"
<br/>
abrir cmd e instalar o Pyinstaller
<br/>
pip install pyinstaller
<br/>
# Usar 
no cmd: python sazonrat.py
